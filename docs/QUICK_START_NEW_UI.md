# Quick Start: New Teacher Dashboard

## 🎉 What's New?

The teacher dashboard now has **5 focused tabs** instead of one overwhelming page!

## 📱 Access

**URL:** https://wonderful-tree-08b1a860f.1.azurestaticapps.net

## 🗂️ The 5 Tabs

### 1. 👥 Monitor (Default)
**What:** Real-time attendance tracking  
**Use when:** During class to see who's present  
**Features:**
- Entry/Exit QR buttons
- Live student count
- Online status indicators
- GPS warnings

### 2. 🔗 Chains
**What:** Chain management  
**Use when:** Chains stall or need management  
**Features:**
- Start/close chains
- Stall alerts (red badge)
- Chain visualization

### 3. 📸 Capture
**What:** Photo verification  
**Use when:** Verifying seating with photos  
**Features:**
- Initiate captures
- Compare snapshots
- View capture history

### 4. 🤖 Quiz
**What:** AI-powered quiz system  
**Use when:** Auto-generating questions during lectures  
**Features:**
- Screen share capture
- Auto question generation
- Runs in background (green ● badge)

### 5. ⚙️ Session
**What:** Session administration  
**Use when:** Ending session or exporting data  
**Features:**
- Session details
- End session
- Export attendance

## 🚀 Quick Workflow

### Starting Class
1. Go to Monitor tab
2. Click "Show Entry QR"
3. Students scan to join
4. Watch real-time updates

### During Class
1. Stay on Monitor tab
2. (Optional) Start quiz on Quiz tab
3. Switch back to Monitor
4. Quiz continues in background

### Ending Class
1. Monitor tab → Show Exit QR
2. Wait for students to scan
3. Session tab → End Session
4. Session tab → Export Attendance

## 💡 Pro Tips

### Multitasking
- Start quiz, then switch tabs
- Green ● badge shows it's still running
- Real-time updates work on all tabs

### Stall Alerts
- Red badge on Chains tab = stalls detected
- Click to resolve
- Badge disappears when fixed

### Tab Memory
- Dashboard remembers your last tab
- Refresh → returns to same tab
- Each session has independent memory

## 🔍 What to Look For

### Good Signs ✅
- 🟢 Live (connection status)
- Green ● on Quiz tab (when active)
- Real-time student updates
- No console errors

### Issues ⚠️
- 🔴 Disconnected (refresh page)
- Red badge on Chains (resolve stalls)
- ⚠️ GPS warnings (check location)

## 📊 Visual Indicators

| Icon | Meaning |
|------|---------|
| 🟢 Live | Real-time updates working |
| 🟢 Online | Student connected |
| ⚪ Offline | Student disconnected |
| 🎯 Holder | Has chain |
| ⚠️ Warning | GPS issue |
| ● ACTIVE | Quiz running |
| (2) | Badge count |

## 🐛 Troubleshooting

**Tab won't switch?**
→ Refresh page

**Quiz not capturing?**
→ Check green ● badge, verify screen share

**No real-time updates?**
→ Check 🟢 Live status, refresh if 🔴

**Stalled chains?**
→ Go to Chains tab, click "Close All Chains"
